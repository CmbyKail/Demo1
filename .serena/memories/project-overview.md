# 情商训练营 (EQ Trainer) 项目概览

## 📋 项目信息
- **类型**: Web 应用程序
- **架构**: 前后端分离
- **前端**: 原生 HTML/CSS/JavaScript (ES Modules)
- **后端**: Python http.server
- **数据存储**: JSON

## 🎯 核心功能
1. 情商场景模拟训练
2. AI 评分和建议
3. 个人成长进度跟踪

## 📁 关键目录
- `js/`: 前端核心逻辑
- `css/`: 样式定义
- `server.py`: 后端服务
- `server_data.json`: 数据存储

## 🚀 启动命令
```bash
python server.py
# 运行在 http://localhost:8000
```

## 📝 开发规范
- **编码格式**: UTF-8
- **前端风格**: 现代 ES Modules, CSS Variables
- **后端风格**: 标准库优先，避免第三方依赖
